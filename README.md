## To Run
* Run `./start.sh` if this is the first time being set up
* Run `docker compose down` and `docker compose up -d` if ports 80 and 3306 do not show up
* Run `mysql -h 127.0.0.1 -P 3306 -u root -p 'travel' < travel-db-dump-2024-fall.sql` to connect the database file
